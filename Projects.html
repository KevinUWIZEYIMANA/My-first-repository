About the Project

SyncPods is a Java-based parental and school-controlled focus app that helps manage audio distribution and screen time for students and learners. It features parental control, license validation, and a distraction-free environment for focused tasks.

Technologies Used

  • Java (Swing for UI)
  • MySQL (for license & user control)
  • Web dashboard (HTML, CSS, JS)
  • Arduino (if applicable for hardware interaction)
